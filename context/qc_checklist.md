# Чек-лист проверки заказа ОТК (ФЕРРУМ)

Этот чек-лист предназначен для финальной проверки изделий перед перемещением заказа в зону отгрузки (на склад).

## 1. Конструктив и Маркировка
- [ ] **Маркировка:** Наличие читаемого номера марки на каждом изделии (согласно КМД).
- [ ] **Геометрия:** Соответствие габаритных размеров (L, B, H) чертежу. Допуски по ГОСТ 23118.
- [ ] **Диагонали:** Проверка прямоугольности рамных конструкций (разность диагоналей).
- [ ] **Комплектность:** Проверка наличия всех деталей в марке (ребра, фасонки, ответные планки).

## 2. Качество подготовки и резки
- [ ] **Кромки:** Отсутствие заусенцев, острых углов и окалины (после плазмы/пилы).
- [ ] **Отверстия:** Соответствие диаметра и межосевых расстояний. Очистка от грата.
- [ ] **Поверхность:** Отсутствие глубоких забоин и следов коррозии под покраску.

## 3. Сварные соединения
- [ ] **Внешний вид:** Швы ровные, без видимых пор, трещин и наплывов.
- [ ] **Брызги:** Полное удаление брызг металла с околошовной зоны.
- [ ] **Зачистка:** Швы зачищены от шлака. В местах, указанных в КМД, швы зашлифованы заподлицо.
- [ ] **Катет шва:** Соответствие катета сварного шва проектным значениям.

## 4. Слесарные и спец. работы
- [ ] **Вальцовка:** (Для обечаек/сальников) Проверка округлости, отсутствие "домика" на стыке.
- [ ] **Резьбы:** Проверка чистоты и целостности резьбовых соединений (если есть).
- [ ] **Притупление:** Снятие фасок на торцах для безопасного монтажа.

## 5. Антикоррозийная защита (Покраска)
- [ ] **Подготовка:** Поверхность обезжирена и очищена от пыли.
- [ ] **Слой ЛКП:** Отсутствие подтеков, непрокрасов ("непрокрасов") и пузырьков воздуха.
- [ ] **Толщина:** (При наличии требований) Проверка толщиномером.
- [ ] **Цвет:** Соответствие RAL, указанному в заказе.

## 6. Системный учет
- [ ] **Трудозатраты:** В ФЕРРУМ внесены фактические минуты по всем операциям.
- [ ] **Количество:** Фактическое количество готовых изделий совпадает с планом в системе.
- [ ] **Фотофиксация:** Сделаны фото готовой продукции перед упаковкой.
