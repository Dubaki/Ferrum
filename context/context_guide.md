### Ключевые решения:
- Сборка + сварка = одна операция (один человек)
- 4 категории габаритов (small/medium/large/xlarge)
- 2 крупногабаритных поста + 2 стандартных
- Сушка блокирует покрасочную камеру
- 8ч смена, без сверхурочных
- Самообучение: plan vs actual → автокоррекция норм (без AI API)
- AI API — только для аналитики и рекомендаций (фаза 6)
